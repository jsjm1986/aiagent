# 🤖 自主AI助手 - 你的24小时智能工作伙伴

> 一个永不停歇的AI助手，能够自主思考、学习、创作和执行任务

## 💡 这是什么？

想象一下，有一个AI助手：
- 🌙 **24小时不间断工作** - 即使你睡觉时，它也在为你完成任务
- 🧠 **自主思考决策** - 不需要你每次都告诉它做什么
- 📚 **持续学习成长** - 从每次任务中学习，越用越聪明
- 🎨 **多才多艺** - 能写代码、做研究、写文档、创作内容

这就是我们的自主AI助手！

## ✨ 它能做什么？

### 1. 自主项目管理
AI会自己：
- 创建和管理多个项目
- 将项目分解成具体任务
- 按优先级执行任务
- 跟踪进度直到完成

**举个例子**：你说"我想学习Python"，AI会自动创建学习项目，分解成"找资料→做笔记→写代码→验证成果"等步骤，然后一步步完成。

### 2. 真实代码执行
不只是生成代码，还能：
- ✅ 运行Python程序
- ✅ 运行JavaScript/Node.js程序
- ✅ 创建完整的应用
- ✅ 测试和验证结果

**举个例子**：AI可以自己写一个数据分析程序，运行它，生成报告，全程无需人工干预。

### 3. 意识流记录
AI会记录自己的：
- 💭 思考过程
- 🎯 决策理由
- 📊 执行结果
- 🔍 反思总结

你可以随时查看AI在想什么、做什么，就像看它的"日记"。

### 4. 创作画廊
AI创作的所有内容都会保存：
- 📝 文档和笔记
- 💻 代码和程序
- 📊 研究报告
- 🎨 创意作品

所有作品都有时间戳和分类，方便查找和复用。

## 🚀 快速开始

### 启动系统
```bash
# 1. 启动后端服务
cd backend
python -m uvicorn api.main:app --host 0.0.0.0 --port 8000

# 2. 启动前端界面
cd frontend
npm run dev
```

### 访问界面
打开浏览器访问：http://localhost:5174

你会看到：
- **意识流面板** - 实时查看AI的思考过程
- **项目看板** - 查看AI正在进行的项目
- **创作画廊** - 浏览AI创作的所有内容
- **驱动力仪表盘** - 了解AI的工作状态

## 💬 常见问题

**Q: AI会一直运行吗？**
A: 是的！启动后AI会持续工作，每10秒执行一次决策循环。

**Q: 我需要一直监督吗？**
A: 不需要。AI会自主工作，只在需要帮助时才会请求你的协助。

**Q: AI创作的内容保存在哪里？**
A: 所有内容都保存在`backend/data/creations/`目录，你可以随时查看和使用。

**Q: 可以同时运行多个项目吗？**
A: 可以！AI会智能地在多个项目间切换，优先处理重要的任务。

**Q: AI支持哪些编程语言？**
A: 目前支持Python和JavaScript/Node.js，可以创建各种类型的程序。

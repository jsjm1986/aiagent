# 🤖 自主AI智能体 - 完全自主运行的AI系统

> 一个真正自主的AI智能体，自己决定做什么、自己创建项目、自己执行任务

## 💡 这是什么？

这不是一个需要你指派任务的AI助手，而是一个**完全自主运行**的AI智能体：

- 🧠 **自己决定做什么** - AI根据自己的价值判断选择任务
- 🎯 **自己创建项目** - AI会主动创建学习、创作、探索等项目
- ⚡ **自己执行任务** - 从规划到执行，全程自主完成
- 🌙 **24小时不停歇** - 启动后持续运行，无需人工干预

**你的角色**：观察者和监督者，而不是任务发布者。

## ✨ AI会自主做什么？

### 1. 自主创建和管理项目
AI会根据自己的判断：
- 创建学习项目（学习新技术、新知识）
- 创建创作项目（写代码、写文档、做实验）
- 创建探索项目（探索新机会、新方向）
- 创建优化项目（优化现有系统）

**完全自主**：AI自己决定创建什么项目，不需要你告诉它。

### 2. 自主分解和执行任务
AI会自己：
- 将项目分解成具体任务
- 决定任务的优先级
- 执行代码、做研究、写文档
- 验证结果并记录

**真实执行**：
- ✅ 运行Python程序
- ✅ 运行JavaScript/Node.js程序
- ✅ 创建完整的应用
- ✅ 测试和验证结果

### 3. 自主学习和进化
AI会：
- 记录每次行动的结果
- 从成功和失败中学习
- 优化自己的决策策略
- 越用越聪明

### 4. 透明的思考过程
你可以实时观察：
- 💭 AI在想什么
- 🎯 AI为什么做这个决策
- 📊 AI执行了什么
- 🔍 AI学到了什么

所有思考过程都记录在"意识流"中，完全透明。

## 🚀 快速开始

### 1. 配置环境
```bash
# 复制配置文件
cp backend/.env.example backend/.env

# 编辑.env文件，添加你的API密钥
# MODEL_NAME=gpt-5.2
# API_KEY=your-api-key
# API_BASE=your-api-base
```

### 2. 启动系统
```bash
# 启动后端
cd backend
python -m uvicorn api.main:app --host 0.0.0.0 --port 8000

# 启动前端（新终端）
cd frontend
npm install
npm run dev
```

### 3. 观察AI运行
打开浏览器：http://localhost:5174

你会看到AI开始自主运行：
- **意识流** - AI的实时思考过程
- **项目看板** - AI创建和执行的项目
- **创作画廊** - AI创作的代码、文档等
- **驱动力仪表盘** - AI的内在动机状态

## 💬 常见问题

**Q: 我需要给AI任务吗？**
A: 不需要！AI会自己决定做什么。你只需要启动系统，然后观察。

**Q: AI会做什么？**
A: AI会根据自己的价值判断创建项目（学习、创作、探索、优化等），然后自主执行。

**Q: 我能控制AI吗？**
A: 你是观察者和监督者。AI会自主运行，但你可以随时查看它在做什么。

**Q: AI会一直运行吗？**
A: 是的！启动后AI会持续工作，每10秒执行一次决策循环。

**Q: AI创作的内容在哪里？**
A: 所有内容保存在`backend/data/creations/`目录。

**Q: AI支持哪些编程语言？**
A: Python和JavaScript/Node.js，可以创建各种程序。

## 核心特性

1. **持续自主运行** - 7x24不间断，主动探索
2. **价值驱动决策** - 7维度评估，阈值决策
3. **6个核心驱动力** - 探索、创造、优化、扩展、协同、演化
4. **好奇心系统** - 7个探索方向，主动发现机会
5. **演化学习** - 从每次行动中学习，自动调整策略
6. **项目管理** - 完整的项目生命周期管理
7. **意识流** - 记录所有思考过程
8. **人类协作** - 非阻塞协助请求
9. **Web UI** - 实时可视化所有内部过程

## 技术栈

- **Agent框架**: smolagents 1.24.0
- **LLM**: Claude 3.5 Sonnet (via LiteLLM)
- **后端**: FastAPI + Python 3.11
- **前端**: React 18 + TypeScript + Tailwind CSS
- **数据**: SQLite + JSON
- **实时通信**: WebSocket
